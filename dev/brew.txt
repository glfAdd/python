设置
vim /opt/homebrew/etc/redis.conf
    redis默认是前台启动, 改为守护进程方式启动, 将 daemonize no 改成yes
    注释掉 bind 127.0.0.1 ::1
    requirepass 设置密码


命令
    启动服务
        brew services start redis
        redis-server /opt/homebrew/etc/redis.conf
    启动客户端(可以用于停止服务)
        redis-cli
    停止服务
        redis-cli shutdown      (正确方式)
        sudo pkill redis-server (强制停止, 会对数据)

************************************************* redis
1. 安装
brew install rabbitmq

2. 安装RabiitMQ的可视化监控插件
/opt/homebrew/Cellar/rabbitmq/3.8.14
sudo sbin/rabbitmq-plugins enable rabbitmq_management

3. 命令
    brew services start rabbitmq

    // 后台启动
    rabbitmq-server -detached
    rabbitmq-server
    rabbitmqctl status
    // 访问可视化监控插件的界面,默认的用户名密码都是guest,登录后可以在Admin那一列菜单内添加自己的用户
    http://localhost:15672
    rabbitmqctl stop 关闭
